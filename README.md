# vlingo-lattice-exchange-camel

[![Javadocs](http://javadoc.io/badge/io.vlingo/vlingo-lattice-exchange-camel.svg?color=brightgreen)](http://javadoc.io/doc/io.vlingo/vlingo-lattice-exchange-camel) [![Build](https://github.com/vlingo/vlingo-lattice-exchange-camel/workflows/Build/badge.svg)](https://github.com/vlingo/vlingo-lattice-exchange-camel/actions?query=workflow%3ABuild) [![Download](https://img.shields.io/maven-central/v/io.vlingo/vlingo-lattice-exchange-camel?label=maven)](https://search.maven.org/artifact/io.vlingo/vlingo-lattice-exchange-camel) [![Gitter chat](https://badges.gitter.im/gitterHQ/gitter.png)](https://gitter.im/vlingo-platform-java/lattice)

The VLINGO XOOM platform SDK implementation of XOOM LATTICE Exchange for Apache Camel.

Docs: https://docs.vlingo.io/vlingo-lattice/exchange

### Installation

```xml
  <dependencies>
    <dependency>
      <groupId>io.vlingo</groupId>
      <artifactId>vlingo-lattice-exchange-camel</artifactId>
      <version>1.5.2</version>
      <scope>compile</scope>
    </dependency>
  </dependencies>
```

```gradle
dependencies {
    compile 'io.vlingo:vlingo-lattice-exchange-camel:1.5.2'
}
```
